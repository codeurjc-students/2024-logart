\chapter*{Resumen}

Hoy en día, la rapidez con la que consumimos cultura provoca un efecto secundario: los matices y las sensaciones que nos transmiten las obras más queridas se terminan diluyendo con el paso de los días. Es difícil retener la frescura de un recuerdo cuando ya estamos saltando al siguiente contenido. Como respuesta a este problema, el presente Trabajo de Fin de Grado propone la creación de ``LogArt: Galería de Recuerdos''. Esta aplicación web funciona como un espacio personal donde los usuarios pueden registrar sus experiencias culturales. De este modo, la herramienta permite guardar anotaciones precisas sobre los libros, videojuegos o canciones que consumen, evitando que esos detalles se pierdan.

El proyecto nació con la idea de construir una plataforma funcional donde primase el factor personal. Se trataba de permitir al usuario organizar sus obras favoritas y, al mismo tiempo, proteger esas reflexiones íntimas y matices subjetivos que definen nuestra relación con la cultura. Para ello, se desarrolló una aplicación Fullstack utilizando el stack MERN (MongoDB, Express.js, React, Node.js). Toda la lógica de negocio y el manejo de datos recaen sobre un \textit{backend} que se comunica de forma segura mediante una API RESTful. Por su parte, el \textit{frontend} se ha diseñado pensando en la comodidad del usuario, con una interfaz flexible que se ajusta sin problemas a cualquier tamaño de pantalla. En LogArt, los usuarios cuentan con un control total sobre sus registros y recuerdos, protegidos siempre por el estándar JWT. Por otro lado, la plataforma ofrece herramientas de gestión para administradores, quienes pueden supervisar la actividad global y consultar datos estadísticos desde un panel centralizado.

El desarrollo se basó en un modelo iterativo e incremental para permitir una evolución constante del software. La calidad del código fue una prioridad central, por ello, se integró una estrategia de pruebas que abarca desde la lógica interna (unitarias e integración) hasta la validación de flujos completos de usuario (\textit{End-to-End}). Con el fin de reforzar estos estándares de calidad, se construyó una infraestructura basada en la cultura DevOps. Utilizando Docker y Docker Compose para el empaquetado del software, asegurando así un entorno estable y reproducible. Este ecosistema se completa con flujos de trabajo en GitHub Actions, que automatizan la validación del código y la entrega de nuevas imágenes mediante un pipeline de CI/CD.

Como resultado, LogArt se presenta como una solución completa y robusta que permite a los usuarios construir un archivo personal y detallado de sus recuerdos culturales. El proyecto no solo cumple con los objetivos funcionales y técnicos propuestos, siendo además una aplicación cada vez más necesaria en el mundo en el que vivimos, sino que también representa una valiosa puesta en práctica de los conocimientos y habilidades adquiridos en el Grado en Ingeniería del Software.
